1.搭建github博客

2.Settings => 选择一个主题保存 就可以访问 meijuna.github.io 了

2.Hexo搭建本地博客

hexo命令

hexo s(server) –> 启动本地服务

hexo clean –> 清除缓存

hexo g (generate) –> 创建网页

hexo d (deploy) –> 同步到github

1.安装node.js + git

以Windows环境安装node.js为例，首先登录node.js官网，选择适合自己的版本进行下载，然后进行安装。

登录git官网选择版本进行下载，基本一路next下去即可轻松完成安装。
